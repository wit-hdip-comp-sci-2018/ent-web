HttpClient

We can start the journey now of integrating the Aurelia SPA application with the Donation-Service HAPI application we have been building.

The first step will be to install a HttpClient library in our SPA. the aurelia-http-client library is a good place to start. First install it:

yarn add aurelia-http-client

Our initial usage will be to retrieve simple json data. Create a new file in our SPA project, containing some sample candidates:

src/static/api/candidates.json

[
  {
    "firstName": "Bart",
    "lastName": "Simpson",
    "office": "President"
  },
  {
    "firstName": "Marge",
    "lastName": "Simpson",
    "office": "President"
  }
]

We rework DonationService now to acquire this component (via dependency injection), and then request this json data via a getCandidates() method we call in the constructor:

src/services/donation-service.ts

import { inject } from 'aurelia-framework';
import { Candidate, Donation } from './donation-types';
import { HttpClient } from 'aurelia-http-client';

@inject(HttpClient)
export class DonationService {
  candidates: Candidate[] = [];
  donations: Donation[] = [];
  paymentMethods = ['Cash', 'Paypal'];

  constructor(private httpClient: HttpClient) {
    httpClient.configure(http => {
      http.withBaseUrl('http://localhost:8080');
    });
    this.getCandidates();
  }

  async getCandidates() {
    const response = await this.httpClient.get('/api/candidates.json');
    this.candidates = await response.content;
    console.log (this.candidates);
  }
}

Restart the application now - and you should see the two candidates preloaded:

Looking carefully at this - first we inject the http client:

@inject(HttpClient)

Then, we initialise the http client with the url of our (SPA) application:

  constructor(private httpClient: HttpClient) {
    httpClient.configure(http => {
      http.withBaseUrl('http://localhost:8080');
    });
    this.getCandidates();
  }

Finally, in getCandidates we generate the http request:

    const response = await this.httpClient.get('/api/candidates.json');

Remember, our SPA is running in a browser (on the client), but the SPA itself will have been downloaded from a server. This is the server we are requesting the json data from - not the Donation-Web hapi application (for the moment). We will take that step in the next lab.

For the moment we will continue with effectively a simulation of the server - integrated into our own application as simple json data.

Event Aggregator - Publish

The Aurelia Event Aggregator is a simple way of facilitating communication between different components. It relies on message types - defined by the application. Once these message types are defined Components can then be either publishers or subscribers to these messages.

Here is a message type to encapsulate a Total Update event:

src/services/messages.ts

export class TotalUpdate {
  total: number;
  constructor(total: number) {
    this.total = total;
  }
}

Because we have centralised the donate feature in the DonationService class - we can refactor it to generate these events when a donation is made:

src/services/donation-service.ts


@inject(HttpClient, EventAggregator)
export class DonationService {

  total = 0;

  constructor(private httpClient: HttpClient, private ea: EventAggregator) {
    ...
  }

  ...

  async donate(amount: number, method: string, candidate: Candidate) {
    ...
    this.ea.publish(new TotalUpdate(this.total));
  }
}

Once we have access to the Event Aggregator (this.ea), we broadcast the new total:

    this.ea.publish(new TotalUpdate(this.total));

Here is the complete DonationService class at this stage:

import { inject } from 'aurelia-framework';
import { Candidate, Donation } from './donation-types';
import { HttpClient } from 'aurelia-http-client';
import { EventAggregator } from 'aurelia-event-aggregator';
import {TotalUpdate} from "./messages";

@inject(HttpClient, EventAggregator)
export class DonationService {
  candidates: Candidate[] = [];
  donations: Donation[] = [];
  paymentMethods = ['Cash', 'Paypal'];
  total = 0;

  constructor(private httpClient: HttpClient, private ea: EventAggregator) {
    httpClient.configure(http => {
      http.withBaseUrl('http://localhost:8080');
    });
    this.getCandidates();
  }

  async getCandidates() {
    const response = await this.httpClient.get('/api/candidates.json');
    this.candidates = await response.content;
    console.log(this.candidates);
  }

  async donate(amount: number, method: string, candidate: Candidate) {
    const donation = {
      amount: amount,
      method: method,
      candidate: candidate
    };
    this.donations.push(donation);
    this.total = this.total + amount;
    this.ea.publish(new TotalUpdate(this.total));
    console.log('Total so far ' + this.total);
  }
}

Event Aggregator - Subscribe

The TotalDonated component can new be refactored to be a subscriber to TotalUpdate event:

src/resources/elements/total-donated.ts

import { inject } from 'aurelia-framework';
import { DonationService } from '../../services/donation-service';
import { bindable } from 'aurelia-framework';
import { TotalUpdate } from '../../services/messages';
import { EventAggregator } from 'aurelia-event-aggregator';

@inject(DonationService, EventAggregator)
export class TotalDonated {
  total = 0;

  constructor(private ds: DonationService, private ea: EventAggregator) {
    this.total = ds.total;
    ea.subscribe(TotalUpdate, msg => {
      this.total = msg.total;
    });
  }
}

The subscription is carried out here:

    ea.subscribe(TotalUpdate, msg => {
      this.total = msg.total;
    });
