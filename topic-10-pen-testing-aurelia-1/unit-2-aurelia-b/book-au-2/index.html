Exercise

Create a new Custom Element to encapsulate the Candidate From:

This will be view two files:

  • /src/resources/elements/candidate-form.ts
  • /src/resources/elements/candidate-form.html

These can contain factored out aspects of the candidates component - and then that component can simplified to use the new element.

Solution

This is the Custom Element View Model:

src/resources/elements/candidate-form.ts

import { bindable } from 'aurelia-framework';
import { Candidate } from '../../services/donation-types';

export class CandidateForm {
  firstName: string;
  lastName: string;
  office: string;
  @bindable
  candidates: Candidate[];

  addCandidate() {
    const candidate = {
      firstName: this.firstName,
      lastName: this.lastName,
      office: this.office
    };
    this.candidates.push(candidate);
    console.log(candidate);
  }
}

it is similar to the Candidates view - but the candidates array is now @bindable - i.e. it is defined and managed elsewhere.

This is the Custom Element View:

src/resources/elements/candidate-form.html

<template>
  <form submit.trigger="addCandidate()" class="ui form stacked segment">
    <h3 class="ui dividing header"> Add a Candidate </h3>
    <div class="field">
      <label>First Name </label> <input value.bind="firstName">
    </div>
    <div class="field">
      <label>Last Name </label> <input value.bind="lastName">
    </div>
    <div class="field">
      <label>Office </label> <input value.bind="office">
    </div>
    <button class="ui blue submit button">Add</button>
  </form>
</template>

We now simplify the Candidates component:

src/views/candidates.ts

import { Candidate } from '../services/donation-types';

export class Candidates {
  candidates: Candidate[] = [];
}

src/views/candidates.html

<template>
  <require from="../resources/elements/candidate-form"></require>
  <require from="../resources/elements/candidate-list"></require>

  <div class="ui stackable two column grid">
    <div class="column">
      <candidate-form candidates.bind="candidates"></candidate-form>
    </div>
    <div class="column">
      <candidate-list candidates.bind="candidates"></candidate-list>
    </div>
  </div>
</template>

This should work as before.

Donate Component Skeleton

Lets try a new Component - to accept donations.

We start with a skeleton

The View Model:

src/views/donate.ts

export class Donate {
}

The View:

src/views/donate.html

<template>
  <div class="ui stackable two column grid">
    <div class="column">
      <p> Donate Form </p>
    </div>
    <div class="column">
      <p> Donations List </p>
    </div>
  </div>
</template>

And then refactor app to require and place this component on the main page:

src/app.ts

<template>
  <require from="views/candidates"></require>
  <require from="views/donate"></require>

  <div class="ui container">
    <section class="ui raised segment">
      <h3 class="ui headder"> Donation </h3>
    </section>
    <div class="ui basic segment">
      <donate></donate>
      <candidates></candidates>
    </div>
  </div>
</template>

This version is empty - just the headings showing up to indicate all parts are present.

Donate Component Implementation

We can fill out the View Model:

src/views/donate.ts

export class Donate {
  amount = '0';
  donations: number[] = [];

  makeDonation() {
    this.donations.push(parseInt(this.amount));
    console.log(this.donations);
  }
}

... and the View:

src/views/donate.html

<template>
  <div class="ui stackable two column grid">
    <div class="column">
      <form submit.trigger="makeDonation()" class="ui form stacked segment">
        <h3 class='ui dividing header'> Make a Donation </h3>
        <div class="grouped inline fields">
          <div class="field">
            <label>Amount</label> <input type="number" value.bind="amount">
          </div>
        </div>
        <button class="ui blue submit button">Donate</button>
      </form>
    </div>
    <div class="column">
      <p> Donations List </p>
    </div>
  </div>
</template>

See if you can identify the references in the above to the View Model (the class):

  • makeDonation()
  • amount

Here is the section to display the donations (put this in the second column):

src/views/donate.html

    <h3 class="ui dividing header"> Donations to Date </h3>
        <table class="ui celled table segment">
          <thead>
            <tr>
              <th>Amount</th>
            </tr>
          </thead>
          <tbody>
            <tr repeat.for="donation of donations">
              <td> ${donation}</td>
            </tr>
          </tbody>
        </table>

donation-list Custom Element

As we did with the list of candidates, we can create a Custom Element to display the donations.

donation-list Custom Element

src/resources/elements/donation-list.ts

import { bindable } from 'aurelia-framework';

export class DonationsList {
  @bindable
  donations : number[];
}

src/resources/elements/donation-list.html

<template>
  <div class="ui stacked segment">
    <h3 class="ui dividing header"> Donations to Date </h3>
    <table class="ui celled table segment">
      <thead>
        <tr>
          <th>Amount</th>
        </tr>
      </thead>
      <tbody>
        <tr repeat.for="donation of donations">
          <td> ${donation}</td>
        </tr>
      </tbody>
    </table>
  </div>
</template>

Now we require and make use of the new element in the donate component:

src/views/donate.html

<template>
  <require from="../resources/elements/donations-list"></require>

  <div class="ui stackable two column grid">
    <div class="column">
      <form submit.trigger="makeDonation()" class="ui form stacked segment">
        <h3 class='ui dividing header'> Make a Donation </h3>
        <div class="grouped inline fields">
          <div class="field">
            <label>Amount</label> <input type="number" value.bind="amount">
          </div>
        </div>
        <button class="ui blue submit button">Donate</button>
      </form>
    </div>
    <div class="column">
      <donations-list donations.bind="donations"></donations-list>
    </div>
  </div>
</template>

The relevant change is:

      <donations-list donations.bind="donations"></donations-list>
