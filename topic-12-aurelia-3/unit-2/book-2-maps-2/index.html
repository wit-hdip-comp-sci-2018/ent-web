Map View Component

We have a map that appears as a custom element on an existing view component (donate). Lets try a completely new view component that only contains a map.

src/views/map.html

<template>
  <div class="ui stacked segment">
    <div id="${mapId}" class="ui embed" style="height:${mapHeight}px; z-index: 0"></div>
  </div>
</template>

src/views/map.ts

import { LeafletMap } from '../services/leaflet-map';
import { DonationService } from '../services/donation-service';
import { inject } from 'aurelia-framework';

@inject(DonationService)
export class Map {
  mapId = 'main-map';
  mapHeight = 600;
  map: LeafletMap;

  constructor(private ds: DonationService) {}

  attached() {
    const mapConfig = {
      location: { lat: 53.2734, lng: -7.7783203 },
      zoom: 8,
      minZoom: 1
    };
    this.map = new LeafletMap(this.mapId, mapConfig, 'Terrain');
    this.map.showZoomControl();
  }
}

This is similar to the map component devised for Lab 12c. However, we can now make it a standalone view. This is achieved by creating a new route in app.ts:

src/app.ts

  ...
      {
        route: 'map',
        name: 'map',
        moduleId: PLATFORM.moduleName('views/map'),
        nav: true,
        title: 'Map'
      },
  ...

Donation Markers

Adjust the map component to display all donations:

src/views/map.ts

import { LeafletMap } from '../services/leaflet-map';
import { DonationService } from '../services/donation-service';
import { inject } from 'aurelia-framework';

@inject(DonationService)
export class Map {
  mapId = 'main-map';
  mapHeight = 600;
  map: LeafletMap;

  constructor(private ds: DonationService) {}

  renderDonations() {
    for (let donation of this.ds.donations) {
      const donationStr = `${donation.candidate.firstName} ${donation.candidate.lastName} €${donation.amount.toString()}`;
      this.map.addMarker(donation.location, donationStr);
    }
  }

  attached() {
    const mapConfig = {
      location: { lat: 53.2734, lng: -7.7783203 },
      zoom: 8,
      minZoom: 1
    };
    this.map = new LeafletMap(this.mapId, mapConfig, 'Terrain');
    this.map.showZoomControl();
    this.map.showLayerControl();
    this.renderDonations();
  }
}

Layer Groups

Leaflet also allows us to define custom layers - and have different sets of markers associated with these layers.

This has been incorporated into the leaflet-map wrapper already. To engage, we can first add a new layer group to appear in the layer control:

src/views/map.ts

    this.map.addLayerGroup('Donations');

Then, when adding a marker, include the layer group name as the last (optional) parameter:

      this.map.addMarker(donation.location, donationStr, 'Donations');

This the complete class:

import { LeafletMap } from '../services/leaflet-map';
import { DonationService } from '../services/donation-service';
import { inject } from 'aurelia-framework';

@inject(DonationService)
export class Map {
  mapId = 'main-map';
  mapHeight = 600;
  map: LeafletMap;

  constructor(private ds: DonationService) {}

  renderDonations() {
    for (let donation of this.ds.donations) {
      const donationStr = `${donation.candidate.firstName} ${donation.candidate.lastName} €${donation.amount.toString()}`;
      this.map.addMarker(donation.location, donationStr, 'Donations');
    }
  }

  attached() {
    const mapConfig = {
      location: { lat: 53.2734, lng: -7.7783203 },
      zoom: 8,
      minZoom: 1
    };
    this.map = new LeafletMap(this.mapId, mapConfig, 'Terrain');
    this.map.showZoomControl();
    this.map.addLayerGroup('Donations');
    this.map.showLayerControl();
    this.renderDonations();
  }
}

All markers will now belong to a 'Donations' group - and can be shown/hidden via the layer control:

We can also set our default layer to be satellite if we like:

    this.map = new LeafletMap(this.mapId, mapConfig, 'Satellite');

Weather Maps

Looking further into this showcase:

  • https://leafletjs.com/examples/layers-control

There are three layers that display current weather information:

  • OpenWeatherMap.Clouds
  • OpenWeatherMap.Pressure
  • OpenWeatherMap.Wind

For instance this is wind:

var OpenWeatherMap_Wind = L.tileLayer('http://{s}.tile.openweathermap.org/map/wind/{z}/{x}/{y}.png?appid={apiKey}', {
  maxZoom: 19,
  attribution: 'Map data &copy; <a href="http://openweathermap.org">OpenWeatherMap</a>',
  apiKey: '<insert your api key here>',
  opacity: 0.5
});

Notice that it requires an API key. This can be secured by signing up with this service:

  • https://openweathermap.org/

In leaflet-map, these layers must be introduced as overlays (as opposed to base layers) and also have an opacity setting:

  overlays = {
    Wind : L.tileLayer('http://{s}.tile.openweathermap.org/map/wind/{z}/{x}/{y}.png?appid={apiKey}', {
      maxZoom: 19,
      attribution: 'Map data &copy; <a href="http://openweathermap.org">OpenWeatherMap</a>',
      apiKey: 'YOURKEYHERE',
      opacity: 0.5
    }),
    Clouds : L.tileLayer('http://{s}.tile.openweathermap.org/map/clouds/{z}/{x}/{y}.png?appid={apiKey}', {
      maxZoom: 19,
      attribution: 'Map data &copy; <a href="http://openweathermap.org">OpenWeatherMap</a>',
      apiKey: 'YOURKEYHERE',
      opacity: 0.5
    }),
    Pressure : L.tileLayer('http://{s}.tile.openweathermap.org/map/pressure/{z}/{x}/{y}.png?appid={apiKey}', {
      maxZoom: 19,
      attribution: 'Map data &copy; <a href="http://openweathermap.org">OpenWeatherMap</a>',
      apiKey: 'YOURKEYHERE',
